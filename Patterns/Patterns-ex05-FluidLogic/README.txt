Patterns-ex05-FluidLogic

Demonstrate the Fluid logic pattern. Here we're using it for a dynamic report facility.

OBS! This project doesn't work for JDK 17 as the JavaScript engine has been removed and 3rd part alternatives
(e.g. Nashorn and Graal-JS) doesn't seem to be working.

See
- dk.universitet.adm.indskrivning.boundary.rest.Rapport: allows javascript to be
executed dynamically.

How to run:
Deploy to the server. A welcome page will be shown.
