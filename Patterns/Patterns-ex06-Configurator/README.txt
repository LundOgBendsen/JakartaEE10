Patterns-ex06-Configurator

Demonstrate the Configurator Java EE Pattern.

See
- dk.universitet.adm.infrastructure.Configurator a producer of configuration
information given by a String
- dk.universitet.adm.indskrivning.boundary.Indskrivning uses the configurator and
gets a message injected.


How to run:
Deploy to the server. A welcome page will be shown. Enter data in the form and
submit. Check the console output. It will contain configuration data.

URL:
http://localhost:8080/Patterns-ex06-Configurator/
