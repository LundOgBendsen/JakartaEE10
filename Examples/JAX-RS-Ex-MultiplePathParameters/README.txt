JAX-RS-Ex-MultiplePathParameters

Demonstrates how to parse and access multiple path parameters, e.g. ...employees/roy-fielding wchich is parsed into first and last name.

Run:
mvn install -P client-build
mvn install -P mvn-deploy
mvn install -P client-build


See:
http://localhost:8080/JAX-RS-Ex-MultiplePathParameters-1.0-SNAPSHOT/resources/employees/roy-fielding

Supports:
GET

Mime types: 
- text/plain
