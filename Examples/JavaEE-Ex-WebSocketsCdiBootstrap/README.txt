JavaEE-Ex-WebSocketsCdiBootstrap

Demonstrates CDI events, web sockets and bootstrap.

How to run:
* mvn clean install -P mvn-deploy
* Go to http://localhost:8080/JavaEE-Ex-WebSocketsCdiBootstrap-1.0-SNAPSHOT/bootstrap.html
* Try clicking the 2 buttons
* Open multiple browser instances and see how events are pushed to all instances using web sockets
