Web-Ex-Http2

Demonstrates how to use http2 push facility.

See Http2Servlet.java.

How to run:
1. Build and deploy with Maven: 'mvn install -P mvn-deploy'.
2. Goto http://localhost:8080/Web-Ex-Http2-1.0-SNAPSHOT/
3. In the browser, press shift+ctrl+i (dev tools), goto the Network-tab.
Inspect and see who has initiated the image resource transfer.
