JAX-RS-Ex-RegExp

Demonstrates how to use regular expressions to match request urls.
See EmplpyeeResource.

Run:
mvn install -P client-build
mvn install -P mcn-deploy
mvn install -P client-build
Run RestClient as Java application.

http://localhost:8080/JAX-RS-Ex-RegExp-1.0-SNAPSHOT/resources/employees/121280-1212
