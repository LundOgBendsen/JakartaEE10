REST - sample, Async processing

Demonstrates how to process server side resources asynchronously. 

Run:
To deploy server side:
mvn install -P mvn-deploy

To run client:
mvn install -P client-build
Reload dependencies
Run RestClient

See:
http://localhost:8080/JAX-RS-Ex-AsyncProcessing-1.0-SNAPSHOT/resources/employees/23
Watch console output from server.

Supports:
GET


Mime types: 
- application/xml
