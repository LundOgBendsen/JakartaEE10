JavaBatch-Ex-Batch

Demonstrates the use of JavaBatch from af Servlet

How to run:
* mvn clean install -P mvn-deploy
* go to url: http://localhost:8080/JavaBatch-Ex-Batch-1.0-SNAPSHOT/JobsServlet
* use the links to operate the application.

See dk/lundogbendsen/batch/JobsServlet.java for how to use Batch API.
