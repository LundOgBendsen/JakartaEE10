JAX-RS-ex-Swagger

Demonstrate basic server side JAX-RS with Swagger (Open API Specification).

See: https://github.com/swagger-api/swagger-core/wiki/Swagger-2.X---Integration-and-Configuration


Run:
mvn install -P mvn-deploy

See:
http://localhost:8080/JAX-RS-ex-SWAGGER-1.0-SNAPSHOT/resources/products
http://localhost:8080/JAX-RS-ex-SWAGGER-1.0-SNAPSHOT/resources/products/1

For Swagger document of this API:
http://localhost:8080/JAX-RS-ex-Swagger-1.0-SNAPSHOT/resources/openapi.json
http://localhost:8080/JAX-RS-ex-Swagger-1.0-SNAPSHOT/resources/openapi.yaml
