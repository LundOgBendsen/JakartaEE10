CDI-Ex-Interceptors

Demonstrates the use of Interceptors

How to run:
mvn install -P mvn-deploy

1. Run the project on a Server (Run as... > Run on Server)
2. The browser opens and shows http://localhost:8080/CDI-Ex-Interceptors-1.0-SNAPSHOT/
3. See console output

Have a look at the artifacts in dk.lundogbendsen.web.
