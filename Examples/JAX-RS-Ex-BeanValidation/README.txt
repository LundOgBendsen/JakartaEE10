REST - sample, Bean validation

Demonstrates use of Bean Validation to validate request parameters en entity bodies.
See the EmployeeResource class.


Build:
mvn install -P client-build

Deploy:
mvn install -P mvn-deploy

Run:
Run RestClient as a Java app.


Operations:
http://localhost:8080/JAX-RS-Ex-BeanValidation-1.0-SNAPSHOT/resources/employees/42

Supports:
GET
POST

Mime types: 
- application/xml
- text/plain
