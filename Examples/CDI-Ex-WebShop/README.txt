CDI-Ex-WebShop

A small webshop powered by CDI.

How to run:
mvn install -P mvn-deploy

1. Run the project on a Server (Run as... > Run on Server)
2. The browser opens and shows http://localhost:8080/CDI-Ex-WebShop-1.0-SNAPSHOT/
3. Use the interface at http://localhost:8080/CDI-Ex-WebShop-1.0-SNAPSHOT/ to explore the example. 

Note:
Database must be running.
