JAXWS2.2-Ex-HandlersClient

Demonstrates the use of JAX-WS handlers. 

How to run:
Run JAXWS2.2-Ex-HandlersEndpoint on Server
Run /src/dk/lundogbendsen/jaxws/Main.java
	
Endpoint address: 
	http://localhost:8080/JAXWS2.2-Ex-HandlersEndpoint/HelloWorld

WSDL address: 
	http://localhost:8080/JAXWS2.2-Ex-HandlersEndpoint/HelloWorld?wsdl
