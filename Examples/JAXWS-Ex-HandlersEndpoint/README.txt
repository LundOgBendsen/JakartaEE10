JAXWS-Ex-HandlersEndpoint

Demonstrates the use of jakarta ws server side handlers.

How to run:
* mvn clean install -P mvn-deploy
* get wsdl from: http://localhost:8080/JAXWS-Ex-HandlersEndpoint-1.0-SNAPSHOT/HelloWorld?wsdl
* start soapui (L:\startSoapUL.cmd)
    * Create new soap project
    * enter wsdl url http://localhost:8080/JAXWS-Ex-HandlersEndpoint-1.0-SNAPSHOT/HelloWorld?wsdl
    * test the service, try 'crap' as a parameter

Endpoint address: 
    http://localhost:8080/JAXWS-Ex-HandlersEndpoint-1.0-SNAPSHOT/HelloWorld

WSDL address: 
	http://localhost:8080/JAXWS-Ex-HandlersEndpoint-1.0-SNAPSHOT/HelloWorld?wsdl
