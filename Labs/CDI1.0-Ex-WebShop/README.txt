CDI1.0-Ex-WebShop

A small webshop powered by CDI.

How to run:
1. Run the project on a Server (Run as... > Run on Server)
2. The browser opens and shows http://localhost:8080/CDI1.0-Ex-WebShop/
3. Use the interface at http://localhost:8080/CDI1.0-Ex-WebShop/ to explore the example. 

Note:
Database must be running.
